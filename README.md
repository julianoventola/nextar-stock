- Implementation of a promotions management dashboard with real-time control and optimistic UI with versioning.

## Features

- Search by product or tag
- Filter by promotions status
- Filter by product category
- Automatic validate stock conflict uppon stock update

## Usage

```bash
# Clone the repository
git clone https://github.com/julianoventola/nextar-stock.git
cd nextar-stock

# Install dependencies
npm install

# Run the app
npm run dev
```
