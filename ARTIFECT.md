# Prompt
Preciso elaborar um Dashboard de Controle de Ofertas. Estas ofertas possuem janelas de tempo estritas, estoque limitado e regras de visibilidade.
Imagine que o estoque de uma oferta é alterado por um usuário do setor de vendas em tempo real, e também alterado por um outro usuário do setor
de gestão enquanto você visualiza a tela.
1. Garanta que o gestor não edite um dado "sujo" (desatualizado)
2. Implemente uma estratégia de Atualização Otimista (Optimistic UI)
para que a interface seja instantânea
3. Evite o conflito de versões de um mesmo item (versionamento de
dados)
Faça a implementação do Dashboard de Controle de Ofertas com as regras indicadas, o projeto deve ser feito em React com Typescript, usar o Zustand para gestão de estados, pensar no design system da aplicação e json-server para os dados, além disso é mandatório o uso de Clean Code, princípios SOLID e a implementação de Testes Unitários e de Integração para as lógicas críticas (ex: validação de estoque, cálculos de desconto e sincronização de estado).
